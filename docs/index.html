Cores com Dicas

Jogo web multiplayer local inspirado em Hues and Cues. Use um tablet como tabuleiro central e os celulares como controles. Sem instalação, sem internet — só Wi-Fi local e diversão.

3 a 10 jogadores ~15 min Wi-Fi local Open source

Construído para a mesa do jantar

Estado 100% no servidor, validações server-side, comunicação em tempo real via WebSocket.

📱

Mestre / Cliente

Tablet roda o tabuleiro em tela cheia; cada celular vê só o que o jogador precisa ver.

⚡

Tempo real

Socket.IO sincroniza dicas, marcadores e pontuação em todas as telas instantaneamente.

📶

Sem internet

Tudo roda em rede Wi-Fi local. Funciona em casa, no acampamento ou em qualquer lugar.

🎨

540 cores únicas

Grid HSL 30×18 cobrindo todo o espectro, com cantos em tons neutros para os extremos.

🔒

Validação no servidor

Dicas com nomes de cor são bloqueadas; toda transição de fase é gerenciada pelo backend.

🪶

Stack mínima

Node.js + Express + Socket.IO. Front em HTML/CSS/JS puros. Sem build, sem framework.

Como funciona

Quatro fases por turno, repetidas até todos terem dado uma dica.

Cor secreta

O servidor sorteia uma cor do grid e envia só para o celular do jogador da vez.

1ª dica (1 palavra)

O jogador da vez digita uma palavra. Os outros tocam no tablet para colocar seu marcador.

2ª dica (até 2)

Mais uma dica e os jogadores podem mover seu marcador ou adicionar um segundo.

Revelação

A cor secreta pulsa no tabuleiro. Pontos sobem na pontuação e o próximo jogador entra.

Pontuação

Quanto mais perto da cor certa, mais pontos.

Coloque para rodar

Pré-requisito: Node.js 18 ou superior.

1. Instale e inicie

# clone o repositório
git clone https://github.com/chicomcastro/cores-com-dicas
cd cores-com-dicas

# instale dependências
npm install

# inicie o servidor
npm start

2. Conecte e jogue

# Tablet (tabuleiro)
http://<ip-local>:3000/board

# Celular (jogador) — escaneie
# o QR code do tablet ou abra:
http://<ip-local>:3000/player

# Todos na mesma rede Wi-Fi.
# Sem internet necessária.
